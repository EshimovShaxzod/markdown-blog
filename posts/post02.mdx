# Markdownda kichik blog post

## Reactjs haqida

React.js — foydalanuvchi interfeyslarini yaratish uchun JavaScript kutubxonasi bo'lib, komponentlar yordamida UI’ni modulli tarzda tuzish imkonini beradi. Har bir komponent mustaqil bo'lib, UI elementlarini qayta foydalanish mumkin bo'lgan qismlarga bo'lish imkonini beradi.

React'ning asosiy kuchi props va state tushunchalarida. Props orqali ma'lumotlar komponentlar orasida uzatiladi, state esa, komponentning ichki holatini bosh

## Reactjs bilan ishlash

React bilan ishlashda, har bir komponent alohida bo'lib, dasturchilarga UI ni oson boshqarish imkonini beradi. Komponentlar o'rtasida ma'lumotlar uzatish va holatlarni boshqarish orqali interaktivlikni ta’minlash mumkin.

Masalan, state yordamida tugma bosilganda hisoblagichni oshiradigan komponentni yaratsak, har safar foydalanuvchi tugmani bossa, holat yangilanadi va UI yangilanib turadi. Bunda useState kabi React kancalari yordam beradi, ular yordamida interaktiv komponentlar qurish juda qulay bo'ladi.
